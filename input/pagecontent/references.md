The following references provide the policy, functional, and technical background used by this guide.

* [eHealth Network Guideline on Hospital Discharge Report - R1.1](https://health.ec.europa.eu/system/files/2024-01/ehn_hdr_guidelines_en.pdf). Defines the Hospital Discharge Report data set used as the starting point for this guide.
* [Xt-EHR logical models](https://www.xt-ehr.eu/fhir/models/1.0.0/en/overview-dischargereport.html). These models refine the eHealth Network Hospital Discharge Report data set and constitute the EHDS reference model at the time of publication of this guide, pending publication of the Article 15 Implementing Act.
* [MyHealth@EU Requirements Catalogue](https://webgate.ec.europa.eu/fpfis/wikis/spaces/EHDSI/pages/1981843035/14.01+Create+the+MyHealth+EU+Hospital+Discharge+Report+content). Provides the functional requirements catalogue for MyHealth@EU cross-border services.
* [MyHealth@EU Hospital Discharge Report FHIR IG](https://fhir.ehdsi.eu/hdr/). Specifies HDR content for MyHealth@EU cross-border services and is derived from this guide.
* [HL7 Europe Core and Base FHIR IG](https://hl7.eu/fhir/base). Provides common European FHIR building blocks reused by this guide.
* [D7.3 Discharge reports: Implementation guides on EEHRxF, functional and technical requirements and specifications for EHR systems](https://www.xt-ehr.eu/wp-content/uploads/2026/05/D7.3.pdf). Xt-EHR deliverable on discharge reports.
* [Regulation (EU) 2025/327 of the European Parliament and of the Council of 11 February 2025 on the European Health Data Space and amending Directive 2011/24/EU and Regulation (EU) 2024/2847 (Text with EEA relevance)](http://data.europa.eu/eli/reg/2025/327/oj). Establishes the European Health Data Space legal framework.
